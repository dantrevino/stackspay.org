title: Quick Start Guide  
description: Get started with StacksPay in 5 minutes
---

Choose your path to get started with StacksPay:

## For Users
- Download a Stacks wallet (Boom, Xverse, Leather)
- Receive a StacksPay link from a merchant
- Click the link to make payment

## For Merchants  
- Set up your Stacks wallet
- Generate StacksPay links using Boom Wallet or SDK
- Share links with customers

## For Developers
- Install the StacksPay SDK: `npm install stacks-pay`
- Create your first payment request
- Integrate with your application
