---
title: '¿Qué es StacksPay?'
description: 'Introducción a StacksPay'
---


StacksPay es una solución de pago descentralizada y potente que te permite enviar y recibir pagos fácilmente en la blockchain de Stacks. Simplifica los pagos generando una URL o código QR que incluye toda la información de pago necesaria. Los usuarios pueden hacer clic en el enlace o escanear el código QR con su wallet de Stacks para iniciar y completar rápidamente un pago.

StacksPay está diseñado para hacer que los pagos basados en blockchain sean fáciles tanto para comerciantes como para usuarios. Ya sea que estés vendiendo productos en línea, aceptando donaciones o solicitando pago por un servicio, StacksPay proporciona una forma directa de manejar transacciones de manera segura y transparente.

## ¿Quién puede usar StacksPay?

StacksPay es versátil y adecuado para:

- **Comerciantes**: ¿Vendes bienes o servicios? Usa StacksPay para solicitar pagos de clientes. Puedes crear un enlace de pago o código QR que los clientes pueden usar para enviar pagos en criptomoneda a tu wallet de Stacks.

- **Creadores y Organizaciones sin Fines de Lucro**: Acepta donaciones con StacksPay. Simplemente genera un enlace de donación y compártelo con tu audiencia. Los donantes pueden elegir la cantidad a enviar.

- **Freelancers y Proveedores de Servicios**: Factura a clientes fácilmente con StacksPay. Crea una solicitud de pago para servicios específicos y comparte el enlace con tu cliente.

- **Usuarios Individuales**: ¿Quieres enviar pagos a amigos, familia u otras personas? StacksPay te permite enviar pagos en criptomoneda de forma rápida y segura.

## Cómo funciona

Usar StacksPay es increíblemente sencillo:

1. **Los comerciantes crean una solicitud de pago**: El comerciante proporciona su dirección de Stacks, establece una cantidad y agrega una descripción. Esta información se convierte en una URL de pago o código QR.

2. **Los usuarios hacen clic en el enlace o escanean el código QR**: El cliente simplemente hace clic en el enlace de pago o escanea el código QR con su wallet de Stacks. La wallet llena automáticamente los detalles del pago.

3. **El pago se realiza en la blockchain**: Una vez que el usuario revisa y confirma el pago, la transacción se procesa en la blockchain de Stacks.

4. **Verificación en tiempo real**: Tanto el comerciante como el cliente pueden verificar la transacción en tiempo real usando la blockchain de Stacks.
