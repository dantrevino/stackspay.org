title: Guía de Inicio Rápido  
description: Comience con StacksPay en 5 minutos
---

Elija su ruta para comenzar con StacksPay:

## Para Usuarios
- Descargue una wallet de Stacks (Boom, Xverse, Leather)
- Reciba un enlace de StacksPay de un comerciante
- Haga clic en el enlace para realizar el pago

## Para Comerciantes
- Configure su wallet de Stacks
- Genere enlaces de StacksPay usando Boom Wallet o SDK
- Comparta enlaces con clientes

## Para Desarrolladores
- Instale el SDK de StacksPay: `npm install stacks-pay`
- Cree su primera solicitud de pago
- Integre con su aplicación
