---
title: 'Was ist StacksPay?'
description: 'Einführung in StacksPay'
---


StacksPay ist eine leistungsstarke, dezentrale Zahlungslösung, die es Ihnen ermöglicht, ganz einfach Zahlungen auf der Stacks-Blockchain zu senden und zu empfangen. Es vereinfacht Zahlungen, indem es eine URL oder einen QR-Code generiert, der alle erforderlichen Zahlungsinformationen enthält. Benutzer können auf den Link klicken oder den QR-Code mit ihrer Stacks-Wallet scannen, um schnell eine Zahlung zu starten und abzuschließen.

StacksPay wurde entwickelt, um blockchainbasierte Zahlungen sowohl für Händler als auch für Benutzer zu erleichtern. Egal, ob Sie Produkte online verkaufen, Spenden annehmen oder eine Dienstleistung in Rechnung stellen, StacksPay bietet eine einfache Möglichkeit, Transaktionen sicher und transparent abzuwickeln.

## Wer kann StacksPay verwenden?

StacksPay ist vielseitig und geeignet für:

- **Händler**: Verkaufen Sie Waren oder Dienstleistungen? Verwenden Sie StacksPay, um Zahlungen von Kunden anzufordern. Sie können einen Zahlungslink oder QR-Code erstellen, den Kunden verwenden können, um Kryptowährungszahlungen an Ihre Stacks-Wallet zu senden.

- **Ersteller und Nonprofits**: Akzeptieren Sie Spenden mit StacksPay. Generieren Sie einfach einen Spendenlink und teilen Sie ihn mit Ihrem Publikum. Spender können den Betrag wählen, den sie senden möchten.

- **Freiberufler und Dienstleister**: Stellen Sie Kunden einfach Rechnungen mit StacksPay. Erstellen Sie eine Zahlungsanfrage für bestimmte Dienstleistungen und teilen Sie den Link mit Ihrem Kunden.

- **Einzelne Benutzer**: Möchten Sie Zahlungen an Freunde, Familie oder andere Personen senden? StacksPay ermöglicht es Ihnen, Kryptowährungszahlungen schnell und sicher zu senden.

## Wie es funktioniert

Die Verwendung von StacksPay ist unglaublich einfach:

1. **Händler erstellen eine Zahlungsanfrage**: Der Händler gibt seine Stacks-Adresse an, legt einen Betrag fest und fügt eine Beschreibung hinzu. Diese Informationen werden in eine Zahlungs-URL oder einen QR-Code umgewandelt.

2. **Benutzer klicken auf den Link oder scannen den QR-Code**: Der Kunde klickt einfach auf den Zahlungslink oder scannt den QR-Code mit seiner Stacks-Wallet. Die Wallet füllt automatisch die Zahlungsdetails aus.

3. **Zahlung wird auf der Blockchain durchgeführt**: Sobald der Benutzer die Zahlung überprüft und bestätigt, wird die Transaktion auf der Stacks-Blockchain verarbeitet.

4. **Echtzeitverifizierung**: Sowohl der Händler als auch der Kunde können die Transaktion in Echtzeit über die Stacks-Blockchain verifizieren.
