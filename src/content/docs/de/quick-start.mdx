title: Schnellstart-Anleitung  
description: Starten Sie mit StacksPay in 5 Minuten
---

Wählen Sie Ihren Weg, um mit StacksPay zu starten:

## Für Benutzer
- Laden Sie eine Stacks-Wallet herunter (Boom, Xverse, Leather)
- Erhalten Sie einen StacksPay-Link von einem Händler
- Klicken Sie auf den Link, um die Zahlung durchzuführen

## Für Händler
- Richten Sie Ihre Stacks-Wallet ein
- Generieren Sie StacksPay-Links mit Boom Wallet oder SDK
- Teilen Sie Links mit Kunden

## Für Entwickler
- Installieren Sie das StacksPay SDK: `npm install stacks-pay`
- Erstellen Sie Ihre erste Zahlungsanfrage
- Integrieren Sie in Ihre Anwendung
