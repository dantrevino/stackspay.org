---
title: 'O que é o StacksPay?'
description: 'Introdução ao StacksPay'
---


StacksPay é uma solução de pagamento descentralizada e poderosa que permite enviar e receber pagamentos facilmente na blockchain Stacks. Simplifica os pagamentos gerando uma URL ou código QR que inclui todas as informações de pagamento necessárias. Os usuários podem clicar no link ou escanear o código QR com sua carteira Stacks para iniciar e completar rapidamente um pagamento.

StacksPay foi projetado para tornar os pagamentos baseados em blockchain fáceis tanto para comerciantes quanto para usuários. Seja vendendo produtos online, aceitando doações ou solicitando pagamento por um serviço, StacksPay fornece uma forma direta de lidar com transações de forma segura e transparente.

## Quem pode usar StacksPay?

StacksPay é versátil e adequado para:

- **Comerciantes**: Vendendo bens ou serviços? Use StacksPay para solicitar pagamentos de clientes. Pode criar um link de pagamento ou código QR que os clientes podem usar para enviar pagamentos em criptomoeda para sua carteira Stacks.

- **Criadores e Organizações sem Fins Lucrativos**: Aceite doações com StacksPay. Simplesmente gere um link de doação e compartilhe com sua audiência. Os doadores podem escolher a quantia a enviar.

- **Freelancers e Prestadores de Serviços**: Fature clientes facilmente com StacksPay. Crie uma solicitação de pagamento para serviços específicos e compartilhe o link com seu cliente.

- **Usuários Individuais**: Quer enviar pagamentos para amigos, família ou outras pessoas? StacksPay permite enviar pagamentos em criptomoeda de forma rápida e segura.

## Como funciona

Usar StacksPay é incrivelmente simples:

1. **Comerciantes criam uma solicitação de pagamento**: O comerciante fornece seu endereço Stacks, define uma quantia e adiciona uma descrição. Esta informação é convertida em uma URL de pagamento ou código QR.

2. **Usuários clicam no link ou escaneiam o código QR**: O cliente simplesmente clica no link de pagamento ou escaneia o código QR com sua carteira Stacks. A carteira preenche automaticamente os detalhes do pagamento.

3. **Pagamento é feito na blockchain**: Uma vez que o usuário revisa e confirma o pagamento, a transação é processada na blockchain Stacks.

4. **Verificação em tempo real**: Tanto o comerciante quanto o cliente podem verificar a transação em tempo real usando a blockchain Stacks.
