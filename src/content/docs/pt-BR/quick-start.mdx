title: Guia de Início Rápido  
description: Comece com StacksPay em 5 minutos
---

Escolha o seu caminho para começar com o StacksPay:

## Para Usuários
- Baixe uma carteira Stacks (Boom, Xverse, Leather)
- Receba um link do StacksPay de um comerciante
- Clique no link para fazer o pagamento

## Para Comerciantes
- Configure sua carteira Stacks
- Gere links do StacksPay usando Boom Wallet ou SDK
- Compartilhe links com clientes

## Para Desenvolvedores
- Instale o SDK do StacksPay: `npm install stacks-pay`
- Crie sua primeira solicitação de pagamento
- Integre com sua aplicação
