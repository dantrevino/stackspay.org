description: Commencez avec StacksPay en 5 minutes
---

Choisissez votre chemin pour commencer avec StacksPay :

## Pour les Utilisateurs
- Téléchargez un portefeuille Stacks (Boom, Xverse, Leather)
- Recevez un lien StacksPay d'un commerçant
- Cliquez sur le lien pour effectuer le paiement

## Pour les Commerçants
- Configurez votre portefeuille Stacks
- Générez des liens StacksPay en utilisant Boom Wallet ou SDK
- Partagez les liens avec les clients

## Pour les Développeurs
- Installez le SDK StacksPay : `npm install stacks-pay`
- Créez votre première demande de paiement
- Intégrez avec votre application
