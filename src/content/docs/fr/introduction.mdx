description: 'Introduction à StacksPay'
---


StacksPay est une solution de paiement décentralisée et puissante qui vous permet d'envoyer et de recevoir facilement des paiements sur la blockchain Stacks. Elle simplifie les paiements en générant une URL ou un code QR qui inclut toutes les informations de paiement nécessaires. Les utilisateurs peuvent cliquer sur le lien ou scanner le code QR avec leur portefeuille Stacks pour initier et compléter rapidement un paiement.

StacksPay est conçu pour rendre les paiements basés sur la blockchain faciles tant pour les commerçants que pour les utilisateurs. Que vous vendiez des produits en ligne, acceptiez des dons ou demandiez un paiement pour un service, StacksPay fournit un moyen direct de gérer les transactions de manière sécurisée et transparente.

## Qui peut utiliser StacksPay?

StacksPay est polyvalent et convient pour :

- **Commerçants** : Vendez-vous des biens ou des services ? Utilisez StacksPay pour demander des paiements aux clients. Vous pouvez créer un lien de paiement ou un code QR que les clients peuvent utiliser pour envoyer des paiements en cryptomonnaie à votre portefeuille Stacks.

- **Créateurs et Organisations à But Non Lucratif** : Acceptez des dons avec StacksPay. Générez simplement un lien de don et partagez-le avec votre audience. Les donateurs peuvent choisir le montant à envoyer.

- **Freelancers et Prestataires de Services** : Facturez facilement les clients avec StacksPay. Créez une demande de paiement pour des services spécifiques et partagez le lien avec votre client.

- **Utilisateurs Individuels** : Voulez-vous envoyer des paiements à des amis, à la famille ou à d'autres personnes ? StacksPay vous permet d'envoyer des paiements en cryptomonnaie rapidement et en toute sécurité.

## Comment ça fonctionne

Utiliser StacksPay est incroyablement simple :

1. **Les commerçants créent une demande de paiement** : Le commerçant fournit son adresse Stacks, définit un montant et ajoute une description. Cette information est convertie en URL de paiement ou code QR.

2. **Les utilisateurs cliquent sur le lien ou scannent le code QR** : Le client clique simplement sur le lien de paiement ou scanne le code QR avec son portefeuille Stacks. Le portefeuille remplit automatiquement les détails du paiement.

3. **Le paiement est effectué sur la blockchain** : Une fois que l'utilisateur examine et confirme le paiement, la transaction est traitée sur la blockchain Stacks.

4. **Vérification en temps réel** : Tant le commerçant que le client peuvent vérifier la transaction en temps réel en utilisant la blockchain Stacks.
